# Online Book Reading System - Frontend

A modern frontend interface for the Online Book Reading System, built using React, Vite, and React Router.

## Technologies Used

- **React 19**: A Javascript library for building user interfaces.
- **Vite 8**: Frontend tooling and dev server.
- **React Router 7**: Declarative routing for React applications.
- **React Pro Sidebar**: Modern sidebar component.
- **Vanilla CSS**: Curated custom styles with dark mode accents.

## Features

- **Dashboard**: View system statistics (total books, categories, and authors).
- **Book Management**: Browse, search, read, and manage books.
- **Author Directory**: Browse and add authors.
- **Category Directory**: Browse and add categories.
- **Bookmarks**: Bookmark books for quick access.
- **User Profile**: Edit user details.
- **Authentication**: Modern, secure login and registration screens.

## Installation & Running

1. Navigate to the `frontend/` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
